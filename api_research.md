# Astronomical Camera APIs

## NASA Astronomy Picture of the Day (APOD)

- **Website:** [https://api.nasa.gov/](https://api.nasa.gov/)
- **GitHub:** [https://github.com/nasa/apod-api](https://github.com/nasa/apod-api)
- **Description:** Provides a daily image or photograph of our universe, along with a brief explanation written by a professional astronomer.
- **Authentication:** Requires an API key.
- **Usage:**
  - `GET https://api.nasa.gov/planetary/apod?api_key=DEMO_KEY`
  - Can fetch images for specific dates or a range of dates.
- **Notes:** This is a great starting point for the project. It's well-documented and provides high-quality images.
